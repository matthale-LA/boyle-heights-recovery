Didn't see the connection row in your Google Sheet? Check these common reasons:

  • Permission Issue: In Apps Script, did you set "Who has access" to Anyone? If left as "Only Myself", sync requests fail.
  • Deployment Issue: When changing scripts, you must deploy a New Deployment. Saving code in the script editor does not automatically update live endpoints.
  • Authorization: Ensure you ran the script manually once in the Apps Script window to grant account drive read/write authorization.

2. Open Extensions > Apps Script.

4. In Apps Script, click Deploy > New Deployment.

5. Click the gear, select Web App, and make sure to configure exactly:

  • Execute as: Me (your email)
  • Who has access: Anyone (not "Anyone with a Google Account")

6. Click Deploy, grant account authorization, copy the Web App URL, and paste it here.
